# 关于
PID控制器是控制领域的经典控制方法。反馈环路由比例、积分、微分三部分。

![PID](/wiki/static/images/pid-diagram.png)


## 比例控制
- 以RC电路为例，假设电源输入电源为u，是控制量，电阻R上电压x为输出量，控制的目标是让x稳定
- 系统方程为
$$
C \frac{d x}{d t} + x = u
$$
- 开环增益为（拉普拉斯变换）
$$
G(s) = \frac{X(s)}{U(s)} = \frac{1}{sC + 1} 
$$
